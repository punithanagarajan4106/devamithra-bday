# 🐼 Project Panda Birthday for DEVAMITHRA

A beautiful, interactive birthday experience featuring cute pandas, heartfelt messages, and precious memories.

## 🌸 Features

- **Loading Screen**: Animated panda with glowing lantern introduction
- **8 Interactive Slides** with unique animations and interactions
- **Glassmorphism Design**: Modern, elegant UI with frosted glass effects
- **Responsive Mobile-First**: Fully responsive on all devices
- **Premium Animations**: Smooth transitions, floating elements, and interactive effects
- **Continuous Background Music**: Soft instrumental that adapts to each slide's mood
- **Photo Gallery**: Curated photo displays with locked allocations
- **Heartfelt Messages**: Personalized birthday letter and tributes

## 📸 Photo Allocation

- **Slide 2**: 4 photos of Devamithra
- **Slide 5**: 3 photos of Devamithra
- **Slide 6**: 1 photo with father
- **Slide 7**: 2 photos (selfie together + walking feet)

## 🎨 Color Palette

- Baby Pink: #FFB6C1

- Lavender: #E6D5F8
- Gold: #FFD700

## 🚀 Setup

1. Clone this repository
2. Add your photos to the `assets/images/` directory following the slide numbers
3. Add background music to `assets/music/background.mp3`
4. Open `index.html` in your browser

## 💖 Slides Overview

## 🎵 Background Music

Place a single continuous instrumental track at `assets/music/background.mp3`. The music will play throughout the experience and complement the mood of each slide.
